# 物件導向程式設計 與 python
- 物件導向程式設計基本觀念
  - 類別(Class)與物件(object)
  - 物件導向三大觀念
    - 封裝
    - 類別與類別的`繼承` ==> 父類別 與 子類別
    - 多型
  - overloading(多載) 與Overridding(覆寫)
- Python模組(module)開發
  - Python內建模組(Python Built-in modules)
- Python套件(package)開發
